### API MODULE QUẢN TRỊ ADMIN & ĐỒNG BỘ ZOHO - ĐẶNG MINH ĐỨC
- GET `/api/admin/users`: Lấy danh sách tài khoản thành viên.
- PUT `/api/admin/users/:id/status`: Khóa/Mở tài khoản.
- POST `/api/admin/canteens/sync`: Đồng bộ dữ liệu sang Zoho CRM.
- GET `/api/admin/dashboard/revenue`: Lấy thống kê tích hợp Zoho Analytics.
- GET `/api/admin/monitor`: Kiểm tra sức khỏe hệ thống hạ tầng.
